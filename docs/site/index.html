Phase 3 — v2 Agentic Graph 재공사

v1.0 은 성공했지만 산출물 자체는 toy 수준이다 — 알고리즘이 입력으로 노출돼 "무엇을 쓸지 알아내는" 핵심 역량이 빠져 있다. v2 는 알고리즘 은닉 · 기법 합성 · 구조 풍부한 hiring-grade 문제를 목표로 토폴로지를 재설계한다 — 단 typed 계약과 검증 해자는 그대로 사수한다.
