⚠️ 알려진 한계
  • • Phase C 직렬화: subprocess + RLIMIT_CPU race(SIGXCPU) 회피 위해 PHASE_C_WORKERS=1 로 직렬 실행. ulimit wrapper 로 병렬 복귀는 후속.
  • • Linux JVM: RLIMIT_AS + JVM 상호작용 불안정 → Java integration test Linux CI 에서 skip.
